**All-in-One Duo** is a high-performance, bulletproof deployment script designed for the extreme network environments of 2026. Built on a "Dual-Core" architecture, it seamlessly switches between **Xray-core (xhttp)** and **Sing-box (Omni-Matrix)**.

## ✨ Key Features
* **Dual-Core Engine:** Deploy Xray and Sing-box suite or individual protocols.
* **VPS Initial Setup:** Auto Swap injection, Ulimit lifting, and deep TCP/BBR optimization.
* **Traffic Guard:** Real-time monitoring via kernel APIs with auto-killswitch functionality.
* **ASN-Aware SNI:** Automatically detects your VPS provider to match authentic SNI targets.
